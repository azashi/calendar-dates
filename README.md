# Calendar Dates

Repo containing examples of calendar built with [dayjs](https://day.js.org/en/)

## Start dev server

- Install dependencies with `yarn` and then run

  ```
  yarn dev
  ```

#### Has **weekly** and **monthly** calendar examples
